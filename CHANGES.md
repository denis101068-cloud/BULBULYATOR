## Технические детали:

- Изображения загружаются с Unsplash (бесплатный сток фотографий)
- Размер изображений оптимизирован (w=400)
- Glide автоматически кэширует изображения для быстрой загрузки
- Поддержка как URL из интернета, так и локальных URI

## Как это работает:

1. При первом запуске создаются 3 товара с URL изображений
2. Glide загружает изображения из интернета
3. Изображения кэшируются для быстрого доступа
4. При добавлении своих товаров можно выбрать фото из галереи
