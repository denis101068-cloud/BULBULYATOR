# Инструкция по работе с фотографиями товаров

## Что изменилось

1. **Вместо эмодзи теперь используются реальные фотографии**
   - В списке товаров отображаются изображения
   - В избранном отображаются изображения
   - При добавлении товара нужно выбрать фото из галереи

2. **Добавление товара с фотографией**
   - Нажмите "Разместить свой товар" в профиле
   - Нажмите кнопку "Выбрать фото"
   - Выберите изображение из галереи телефона
   - Заполните название, описание и цену
   - Нажмите "Разместить товар"

3. **Разрешения**
   - При первом выборе фото приложение запросит разрешение на доступ к галерее
   - Разрешите доступ для возможности выбора фотографий

## Технические детали

- Используется `ACTION_OPEN_DOCUMENT` для выбора изображений
- Сохраняется постоянный доступ к URI изображения
- Поддержка Android 13+ (READ_MEDIA_IMAGES) и более старых версий (READ_EXTERNAL_STORAGE)
- ImageView с `scaleType="centerCrop"` для правильного отображения

## Тестовые товары

При первом запуске создаются 5 тестовых товаров с изображением по умолчанию.
Вы можете добавить свои товары с реальными фотографиями.
